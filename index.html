Gatsby & Sanity

There are a lot of benefits that come from building websites in modern techstacks. By using Gatsby we allow our designers to have access to native page transitions, while still having an extremely performant website. Sanity is a headless CMS that comes with a built-in dashboard that makes it easy to edit content in the user's site.

Questions You Might Have

Is it secure?

Gatsby compiles all the code and turns it into static HMTL, CSS and JS files. If someone tries to hack your site, all they will see is those files. Your data is safe.

As for where all of the data is stored, Sanity encrypts your data at rest and in transit. Basically your data is always encrypted whether it is sitting in Sanity or whether it is being called into Gatsby.

Could we integrate a store or the videos?

You can build custom e-commerce sites using React, since it is extremely easy to add the stripe api and work with taking secure payments in Gatsby. Also since everything is built in Javascript it is simpler to work with the JW Player API.

  • shopflamingo.com
  • ca.braun.com
  • propapanda.eu

Are certain posts able to have different layouts than the others?

Gatsby uses a query language called GraphQL, which allows the developer to choose exactly what data they want and only that. Using GraphQL, the developer can specify what they want for a specific layout, so there can be multiple different ways to create unique layouts while still keeping the ability to dynamically create posts and pages.
